# Random Number Game

This is a React application for the Random Number Game.

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

## Running the Game

1.  Start the development server:
    ```bash
    npm run dev
    ```
2.  Open your browser at the URL shown (usually `http://localhost:5173`).

## Building for Production

1.  Build the app:
    ```bash
    npm run build
    ```
2.  Preview the build:
    ```bash
    npm run preview
    ```
