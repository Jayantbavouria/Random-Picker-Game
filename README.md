# Random Picker Game

A feature-rich web application for random number selection with customizable entries and dual game modes.

## Table of Contents

- [Overview](#overview)
## Overview

Random Picker Game is an interactive React-based application that generates random selections from a user-defined range of numbers. Each number can be mapped to custom strings, making it perfect for various use cases including raffles, team assignments, classroom activities, and decision-making tools.

## Features

- **Flexible Range Selection**: Define any numeric range (X to Y)
- **Auto-populated Defaults**: Entries automatically filled with corresponding number values
- **Custom Mapping**: Associate each number with personalized text
- **Dual Game Modes**:
  - Default Mode: Allows number repetition
  - No-Repeat Mode: Each number selected only once
- **Visual Tracking**: Real-time display of remaining and picked entries
- **Smooth Animations**: Engaging spin animation with result celebration
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Progress Indicators**: Track remaining entries in No-Repeat mode
- **Reset Functionality**: Quickly restart with new configuration
